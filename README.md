# 🚦 Smart Traffic Big Data Project

Proyek ini merupakan implementasi sederhana analisis dan prediksi traffic menggunakan PySpark dan Streamlit.

## 📌 Fitur
- Data processing menggunakan PySpark
- Penyimpanan data dalam format Parquet
- Dashboard interaktif menggunakan Streamlit
- Prediksi jumlah kendaraan menggunakan Linear Regression

## 🛠️ Teknologi
- Python
- PySpark
- Streamlit
- Plotly
- Scikit-learn

## ▶️ Cara Menjalankan

### 1. Aktifkan virtual environment
```bash
source venv/bin/activate

### 2. Jalankan data processing
python main_uts.py

### 3. Jalankan dashboard
streamlit run dashboard.py

📊 Output
Folder output/ berisi data hasil processing (parquet)
Dashboard menampilkan:
Total kendaraan
Grafik tren traffic
Prediksi kendaraan
